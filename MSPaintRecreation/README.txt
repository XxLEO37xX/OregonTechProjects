# CST238 Assignment 1 - Paint Clone

## Description

A simple C# console application that recreates some basic functionality of Microsoft Paint using text-based graphics.

Users can draw on a console-based canvas using the mouse to make some simple shapes with a wide variety of colors.

## Features

- Draw on a console canvas
- Create basic shapes
- Clear the canvas
- Save and/or load drawings
- Simple text-based user interface

## Technologies

- C#
- .NET
- Visual Studio

## Requirements

- Visual Studio 2022 or later
- .NET SDK (version used for the project)

## Running the Project

1. Clone or download this repository.
2. Open `CST238Assign1.sln` in Visual Studio.
3. Press **F5** or click **Start**.
4. Follow the on-screen menu prompts.
